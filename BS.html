From Thesis to Tool 🔧

This simulator wasn't built in a day. Like I mentioned at the beginning, it started as my thesis project — titled "Haavoittuvuuksien jäljillä", which means "Tracing Vulnerabilities" in English.

It started as a small idea — just a simulator and maybe a single HTML page with a few commands and theory. But somewhere between keeping it simple and doing justice to the theory I'd gathered, it grew. So yes, it became a bit more than I first imagined — and yes, built on more coffee-fueled sessions than I'd like to admit.

🕵
If you're curious about how this simulator was built — or want to dive into the theory of cybersecurity and ethical hacking (or learn Finnish the hard way) — check out the thesis on Theseus.fi. Just search for Haavoittuvuuksien jäljillä, written by Marika Salonen — hopefully available in spring 2025.
